SRSF Project Info:

Learning vocabulary in a second is a tedious process, and for Japanese it's especially challenging due to each word having a pitch accent that can be tricky for learners to perceive. Existing tools focus on dictionary lookups or visual/audio cues to aid learners during language immersion. Studies have shown a correlation between touch and increased retention during the acquisition of new vocabulary, but no widely used tool aimed at streamlining immersion using this has been created. We developed a tool that maps Japanese pitch accent onto a tactile system that allows learners to feel different pitch patterns with their hands. It does this by moving mechanical cams using stepper motors representing pitch through high/low pegs. We have an accompanying website where you can scan the desired word. Preliminary testing suggests that using the device could improve retention. This project demonstrates a novel approach to multisensory language learning.
